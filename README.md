# 🧠 Ultimate Focus Tracker

A creative, animated, and mobile-friendly **Focus Tracker** web app that helps improve productivity by tracking focus sessions, distractions, and daily focus score.

## Features
- 🌊 Animated wave timer  
- 📊 Daily focus report (time & distractions)  
- 🔔 Alerts + sound when session ends  
- 🌙 Dark/Light mode toggle  
- 📱 Mobile-friendly responsive UI  
- ☁️ Persistent data using localStorage (simulated cloud save)  

## How to Use
1. Open `index.html` in your browser  
2. Click **Start** to begin a focus session  
3. Click **Got Distracted** if distracted  
4. View your **daily report** in the app  
5. **Reset** button clears session and distractions  
6. Use **Toggle** button to switch Dark/Light mode  

## Live Demo
You can host your app for free with GitHub Pages:
